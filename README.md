
###Reto Automatización Página Despegar

###Contexto
El objetivo de este reto técnico es colocar en práctica los conocimientos de Automatización en Java

###Descripción
Buscar en la página de Despegar.com los 7 precios más baratos de tiquetes entre Medellín y Cartagena entre el 01 y el 29 de septiembre para 2 pasajeros, guardarlos en un archivo de excel y pintar de verde el precio más económico


###Resumen Codificación
Framework: Cucumber
Patrón de Desarrollo: Page Object
Práctica Automatización: BDD con Gherkin
Dependencias: Gradle
IDE: Eclipse por Java Developers Version: Oxygen.1a Release (4.7.1a)
Versionamiento: GITHUB


###Navegador y versión utilizada
Google Chrome Versión 66.0.3359.181 (Build oficial) (64 bits)


###Repositorio 
https://github.com/yeniferciro/DespegarReto.git


###Conclusiones 
- Es necesario afianzar conocimientos en Java, de modo que sea más ágil la solución de los errores que se generan al realizar las automatizaciones
- Al dar solucionar a un planteamiento debe empezarse de la forma más sencilla e ir mejorando el código.
- Para interiorizar conceptos y patrones se debe estar en constante práctica. "La práctica hace al maestro!"
